# NQDrive - Welcome Page

## Overview

Buat halaman utama (Welcome Page) sederhana untuk **NQDrive**.

Halaman ini hanya ditampilkan sebelum pengguna login.

Fokus pada desain yang bersih, modern, profesional, dan memberikan kesan premium tanpa terlalu banyak informasi.

Inspirasi desain:

* Vercel
* Dropbox
* Cloudflare
* GitHub
* Linear

---

# Tech Stack

* React + Vite
* Tailwind CSS
* Shadcn UI
* Motion (Framer Motion)
* Lucide React

---

# Motion

Seluruh halaman menggunakan Motion.

Animasi:

* Smooth Page Transition
* Fade Up
* Fade In
* Stagger Animation
* Hover Lift
* Hover Scale
* Hover Shadow
* Navbar Blur on Scroll
* Button Hover Animation

Durasi animasi:

0.2–0.4 detik

Gunakan easing yang halus.

Jangan menggunakan animasi berlebihan.

# Design Goal

Welcome Page harus memberikan kesan profesional sejak pertama kali dibuka.

Tampilan bersih, ringan, cepat, dan modern.

Animasi harus terasa halus menggunakan Motion tanpa mengganggu performa atau pengalaman pengguna.

Gunakan komponen Shadcn UI dan icon Lucide React secara konsisten di seluruh halaman.
